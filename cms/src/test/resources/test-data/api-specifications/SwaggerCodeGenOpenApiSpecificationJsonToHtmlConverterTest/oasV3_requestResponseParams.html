/op-with-params-with-all-min-max-exlusives
Try this API

Verifies that inclusive and exclusive status of min and max values is indicated correctly.

Request

Headers
Name Description
param-with-min-inclusive-max-exclusive

String

Verifies that inclusive min and exclusive max are marked as such.

Maximum: 10.1 (exclusive)

Minimum: -10.11 (inclusive)

param-with-min-exclusive-max-inclusive

String

Verifies that exclusive min and inclusive max are marked as such.

Maximum: 10.11 (inclusive)

Minimum: -10.12 (exclusive)

param-with-min-max-inclusive-by-default

String

Verifies that min and max are marked as inclusive by default.

Maximum: 10.14 (inclusive)

Minimum: -10.13 (inclusive)
